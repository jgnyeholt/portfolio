About

Hello! Thank you for visiting my website! My name is Jonathan and I am a web developer living in Greenville, South Carolina. Keep reading to hear about my approach to building websites, or scroll down to check out some of my recent projects.

When I start a web project, I consider the end goal of having a website that is intuitive, functional, and engaging. To achieve an intuitive design, I structure a series of steps that I will use to help the user get from point A to point B throughout the website. Consequently, there are many aspects to consider to ensure the structure works well and loads quickly on a range of devices, from phones to computers. Along the way, I strive to make the website engaging as users hover over different elements on the page, click buttons, and more.

My Tools:

  • HTML/CSS(Sass)
  • JavaScript
  • jQuery
  • AJAX
  • PHP/SQL
  • GIMP(photo editing and background design)

The programming tools mentioned here represent only a handful of the many tools developers use to create websites that are both appealing and useful. I look forward to expanding my skill set by learning new languages, frameworks, and libraries to create a better user experience for every website I create.
