About

Hello! I'm Jonathan! I enjoy web development and building sites. Keep reading to learn about my approach and the tools I use.

A mobile responsive website is imperative to reach an increasingly mobile audience. Therefore, I design with a mobile-first approach, incorporating various strategies to keep the website loading fast and easily readable on phones. This includes CSS/Sass media queries to change dimensions and sizes of text, areas around the text, and pictures. I use SVGs and .png photo files to keep the file sizes to a minimum, using GIMP (a Photoshop alternative) to scale down larger pictures.

Designing the user experience is one of my favorite aspects of web design. This ranges from buttons that change when you hover or click, modal popup boxes, and even the navigation bar on the right side of this screen. I enjoy using JavaScript and jQuery alongside CSS, allowing for expanded possibilites and features such as search functionality.

For websites that need more than just a good look, I use PHP and C#/.Net to provide access to databases where information can be stored, updated, or deleted. These languages can turn websites into handy tools that can be used to organize addresses, recipes, or serve as a blog.

The coding tools I have mentioned here represent only a handful of the many tools used to create websites that are not only good looking but are also useful. I look forward to expanding my "toolbox" by learning new languages, frameworks, and libraries to create a better user experience for every website I create.
